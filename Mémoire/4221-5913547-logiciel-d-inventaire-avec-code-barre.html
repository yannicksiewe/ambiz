Le logiciel d'Inventaire avec Code Barre (ICB) vous permettra de simplifier votre saisie et limiter les erreurs dues à une ressaisie.

- Edition des étiquettes code barre pour chacun des produits qui doit être inventorié.
- L'opérateur s'identifie une fois en début d'inventaire.
- Identification du produit avec son code barre et saisi de la quantité comptée sur le clavier.

Le transfert des informations se fait à l'aide du puit de vidage qui est connecté sur un ordinateur. A tout moment, l'opérateur peut visualiser les informations saisies sur l'écran du stylo autonome.

L'inventaire peut se faire à partir de plusieurs stylos simultanément.

Le logiciel est aussi réalisé avec : Terminal laser portable MW F734

- Alimentation : Batterie NiCd 250mAh
- Mémoire : 512Ko pour les programmes et 1024Ko pour les données (max 15 fichiers)
- Ecran : LCD graphique rétro-éclairé de 4×16 caractères
- Distance de lecture : Entre 7 et 70 cm
- Logiciels : Programme de transfert et de configuration entre PC et Terminal
- Code à barres : 15 types dont EAN/UPC, 2/5, code 39, code 128
- Autonomie : Environ 1500 scans
- Dimensions : 5,6×3,2×16,5
- Poids : 184 g

Imprimante Transfert Thermique MW X1000+ :

- Impression codes à barres : Multicodes : UPC/EAN, 2/5, code 39, code 128, PDF417…
- Mémoire de 512 Ko
- Logiciels : Langage d’imprimante compatible Datamax Drivers Windows NT4, 9x, 2000 et XP
- Logiciel d’impression graphique sous Windows Argobar
- Dimension d’impression : Largeur maximale : 104 mm
- Hauteur maximal : 1143 mm
- Interfaces : Série et parallèle
- Dimensions I×P×H : 25,0×41,0×26,0 cm
